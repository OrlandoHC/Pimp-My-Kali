Pimp My Kali

Pimp My Kali es un Script Automatizado para la Personalización del Entorno y El Funcionamiento de Kali Linux posterior a su Instalación.

Acerca De...

Pimp My Kali es un script no oficial desarrollado en Bash que recopila e instala modificaciones y aplicaciones a Kali Linux 2.0 posterior a su instalación. Pimp My Kali esta enfocado para cualquier usuario (novato o experto) ya que no hay nada mejor que un script personalice nuestro entorno para hacerlo mas ameno siendo así live cd o instalación; pmk.sh proporciona las herramientas imprescindibles así como las modificaciones necesarias para desempeñar las tareas cotidianas al utilizar nuestra distro de seguridad.
